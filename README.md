# Machine Learning
This repository contains implementations and experiments based on TU Berlin's *Machine Learning 1* course, with a focus on the mathematical and algorithmic foundations of machine learning.

## Topics Covered
- Bayesian decision theory and probabilistic classification
- Maximum likelihood and Bayesian parameter estimation
- Dimensionality reduction (PCA via SVD and power iteration)
- Clustering methods (weighted k-means)
- Linear discriminant analysis (Fisher LDA)
- Gaussian processes for regression
- Kernel methods and Support Vector Machines
- Statistical estimation
- Neural networks
- Machine learning on real datasets

## Structure
Each folder corresponds to one exercise and includes:
- implementation (NumPy, Matplotlib, Scipy, scikit-learn, PyTorch)
- experiments and evaluations
- results and observations

## Exercises
- Exercise 01: Bayes Decision Theory
- Exercise 02: Maximum Likelihood & Bayes Parameter Estimation
- Exercise 03: PCA with SVD and Power Iteration
- Exercise 04: Weighted K-Means Clustering
- Exercise 05: Fisher Linear Discriminant
- Exercise 06: Gaussian Processes
- Exercise 08: Kernel Support Vector Machines, CVXOPT
- Exercise 09: The James-Stein Estimator
- Exercise 10: Training a Neural Network from scratch
- Exercise 11: Training Neural Networks on the Breast Cancer dataset

## Tech stack
Python (NumPy, SciPy, scikit-learn, Matplotlib, PyTorch)
